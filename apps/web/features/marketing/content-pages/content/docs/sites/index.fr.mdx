Un site est une collection de pages, de mises en page et de documents publiés en tant qu'unité sous une seule URL.

```
https://votre-equipe.baseblocks.dev/votre-site
```

Une équipe peut avoir autant de sites que nécessaire. Patterns courants :

- Un site par produit ou service (`/produit-a`, `/produit-b`)
- Un site par audience (`/interne`, `/integration`, `/partenaires`)
- Un site par projet ou trimestre

## Principes de base du site

### Créer un site

Depuis le tableau de bord de l'équipe, cliquez sur **Nouveau site**. Entrez un nom et un identifiant. L'identifiant devient le segment de chemin dans l'URL.

Les identifiants sont en minuscules et séparés par des tirets. Ils doivent être uniques au sein de l'équipe. Vous pouvez modifier l'identifiant ultérieurement dans les paramètres du site — consultez la [page des paramètres](/docs/sites/settings) pour les implications.

### Structure des URL de site

```
https://[identifiant-equipe].baseblocks.dev/[identifiant-site]/[identifiant-page]
```

Exemple :

```
https://acme.baseblocks.dev/ingenierie/integration
```

Où :
- `acme` est l'identifiant de l'équipe
- `ingenierie` est l'identifiant du site
- `integration` est l'identifiant de la page

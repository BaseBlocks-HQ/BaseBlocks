title: Branding
description: Configure the name and logo displayed in your published site's sidebar.
icon: Palette
---

Published sites use one consistent shell: a page sidebar and a transparent content header. Visual themes and navigation layouts are not configurable.

Open **Site Settings** in the editor sidebar to choose whether the site name and logo appear in the published sidebar.

## Site name

The site name is managed in Site Settings. Disable **Show site name** to keep it out of the published sidebar.

## Logo

Upload or replace the site's logo in Site Settings. Disable **Show logo** to hide it without deleting the uploaded asset.

## Fixed site shell

The sidebar always contains page navigation. The header always exists in the content column and contains public actions such as theme mode and, when enabled, search.

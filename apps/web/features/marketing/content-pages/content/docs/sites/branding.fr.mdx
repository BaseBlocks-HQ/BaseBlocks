Les sites publiés utilisent une structure unique et cohérente : une barre latérale pour les pages et un en-tête transparent dans la zone de contenu. Les thèmes visuels et les dispositions de navigation ne sont pas configurables.

Ouvrez **Paramètres du site** dans la barre latérale de l'éditeur pour choisir si le nom et le logo apparaissent dans la barre latérale publiée.

## Nom du site

Le nom se gère dans les paramètres du site. Désactivez **Afficher le nom du site** pour le masquer dans la barre latérale publiée.

## Logo

Importez ou remplacez le logo dans les paramètres du site. Désactivez **Afficher le logo** pour le masquer sans supprimer le fichier importé.

## Structure fixe du site

La barre latérale contient toujours la navigation des pages. L'en-tête existe toujours dans la zone de contenu et contient les actions publiques, notamment le thème et, lorsqu'elle est activée, la recherche.

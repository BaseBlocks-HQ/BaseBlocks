Chaque site a ses propres paramètres de personnalisation, indépendants des autres sites de l'équipe. Les modifications s'appliquent immédiatement au site publié — vous n'avez pas besoin de publier pour que les modifications de personnalisation prennent effet.

Accédez à ces paramètres depuis le rail flottant sur le bord gauche de l'éditeur :
- **Personnalisation** (icône palette) — couleur d'accent, arrière-plan de l'en-tête et rayon de bordure
- **Navigation** (icône organigramme) — style de navigation

## Couleur d'accent

La couleur interactive principale utilisée sur l'ensemble du site — liens, boutons, états actifs et anneaux de focus.

Définissez-la comme valeur hexadécimale (ex. `#3b82f6`) ou une valeur HSL. L'éditeur affiche un aperçu en direct au fur et à mesure que vous tapez.

La couleur d'accent se propage dans le site via des propriétés personnalisées CSS (`--accent`). Tous les éléments interactifs l'héritent automatiquement.

## Arrière-plan d'en-tête

La couleur d'arrière-plan de l'en-tête de navigation du site. Par défaut, c'est la couleur d'arrière-plan du site.

Définir un arrière-plan d'en-tête distinct aide à établir une hiérarchie visuelle, notamment sur les sites avec une zone de contenu claire et une barre de navigation de marque plus sombre.

## Rayon de bordure

Contrôle l'arrondi des éléments d'interface — cartes, boutons, champs de saisie et conteneurs.

| Réglage | Valeur | Apparence |
|---|---|---|
| Aucun | `0px` | Coins nets, interface plate |
| Petit | `4px` | Légèrement arrondi |
| Moyen | `8px` | Arrondi standard (par défaut) |
| Grand | `12px` | Bien arrondi |
| Complet | `9999px` | Boutons en forme de pilule et conteneurs entièrement arrondis |

## Style de navigation

Contrôle comment la navigation est rendue sur le site public. Se configure depuis le panneau **Navigation** dans le rail.

| Style | Description |
|---|---|
| Barre latérale | Arborescence des pages verticale dans une barre latérale gauche persistante |
| Navigation haute | Menu horizontal dans l'en-tête avec des menus déroulants pour les pages imbriquées |
| Barre d'onglets | Barre d'onglets horizontale sous l'en-tête |

Le style de navigation affecte uniquement le site publié. L'éditeur affiche toujours le chrome d'éditeur par défaut.

The editor opens when you enter a site. It is a block-based writing experience presented inside the same polished site workspace used for pages, publishing, and settings.

## Editor areas

### Header

The top bar contains the site switcher, sharing controls, published-site shortcut, and publish or unpublish action. Editing saves automatically.

### Tool sidebar

A floating sidebar on the left contains editor tools:

- **Pages** — the full page tree. Add, rename, nest, and reorder pages.
- **Site Settings** — identity, favicon, published-site options, visibility, and custom domains.

Add blocks from the slash menu directly in the canvas.

### Page editor

The canvas is the editable document. Type `/` to insert blocks, select text for formatting controls, and drag supported blocks to reorganize the document.

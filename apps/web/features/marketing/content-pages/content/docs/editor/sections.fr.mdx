Une page est une liste ordonnée de sections. Une section contient une ou deux colonnes, et une colonne peut contenir plusieurs blocs.

## Types de sections

| Type | Colonnes | Région |
|---|---:|---|
| Une colonne | 1 | Contenu principal |
| Deux colonnes | 2 | Contenu principal |
| Contenu latéral | 1 | Partie droite |

L'espacement est un bloc, et non une ligne structurelle spéciale.

## Modifier les sections

Choisissez **Sections** dans le rail de l'éditeur pour en ajouter une. Faites glisser sa poignée pour la réorganiser dans la région principale ou latérale. La suppression d'une section supprime également ses colonnes et ses blocs.

Sélectionnez une colonne avant de choisir un bloc dans le rail. Les blocs peuvent être réorganisés dans une colonne ou déplacés directement vers une autre colonne.

## Comportement mobile

Les sections à deux colonnes se réduisent à une colonne sur les petits écrans. Les sections latérales sont affichées après le contenu principal.

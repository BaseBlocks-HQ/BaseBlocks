Les sections, les régions principales et latérales, les onglets et les emplacements en colonnes appartiennent au modèle de l'Éditeur classique. Le nouvel éditeur utilise un document ordonné et prend en charge les colonnes comme nœuds du document.

<Callout type="info">
  Lorsqu'une page classique s'ouvre dans le nouvel éditeur, les sections sont converties dans l'ordre, les sections à deux colonnes deviennent des nœuds en colonnes et le contenu latéral est placé dans le flux. Les onglets sont aplatis dans l'ordre du document. Utilisez **Éditeur classique** pour consulter la mise en page d'origine pendant la bêta.
</Callout>

## Types de sections

| Type | Colonnes | Région |
|---|---:|---|
| Une colonne | 1 | Contenu principal |
| Deux colonnes | 2 | Contenu principal |
| Contenu latéral | 1 | Partie droite |

L'espacement est un bloc, et non une ligne structurelle spéciale.

## Modifier les sections

Ces contrôles restent disponibles uniquement dans l'Éditeur classique. Le nouvel éditeur masque les panneaux Sections et Blocs du rail ; utilisez son menu slash et ses contrôles de document.

Sélectionnez une colonne avant de choisir un bloc dans le rail. Les blocs peuvent être réorganisés dans une colonne ou déplacés directement vers une autre colonne.

## Comportement mobile

Les sections à deux colonnes se réduisent à une colonne sur les petits écrans. Les sections latérales sont affichées après le contenu principal.

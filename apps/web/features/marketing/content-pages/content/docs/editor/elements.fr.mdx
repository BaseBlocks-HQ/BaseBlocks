Un élément est un bloc dans le document de la page. Tapez `/` dans l'éditeur pour ouvrir le menu d'insertion. La barre de sélection donne accès à la mise en forme du texte.

## Blocs

### Blocs de contenu général

Unités de contenu à usage général.

**Titre** — Un titre ou en-tête de section. Supporte H1 à H6. Utilisé pour les titres de pages et les séparations de sections.

**Paragraphe** — Bloc de texte simple. Supporte la mise en forme inline basique (gras, italique, liens).

**Listes et tâches** — Listes à puces, numérotées, de tâches, repliables et imbriquées.

**Encadré** — Une boîte de notice mise en évidence. Supporte les variantes info, avertissement, erreur et succès. Utilisez-le pour les notes importantes, conseils ou alertes.

**Code** — Un bloc de code avec coloration syntaxique. Supporte plusieurs langages. Inclut un bouton de copie.

**Séparateur** — Une règle horizontale. Utile pour séparer visuellement les sections d'une page.

**Espaceur** — Espace vertical entre des blocs voisins.

**Sous-page** — Intègre une carte ou une liste avec un lien vers une page enfant. Utile sur les pages parentes qui servent de pages d'index pour une section.

**Répertoire** — Affiche une liste de pages ou sous-pages. Utile pour construire des hubs de navigation ou lister toutes les pages d'une section.

**Organigramme** — Affiche un diagramme à partir de la syntaxe [Mermaid](https://mermaid.js.org). Collez du code compatible Mermaid pour générer des organigrammes, des diagrammes de séquence et plus encore.

**Arbre de décision** — Un élément de branchement interactif pour présenter des chemins de contenu conditionnels. Les utilisateurs cliquent sur les choix pour atteindre différents résultats.

## Groupes de contenu plus larges

### Sections

Blocs de contenu plus larges pour la navigation et la découverte.

**Liens rapides** — Une grille configurable de liens avec icônes. Utilisez pour les hubs de navigation, les listes de ressources ou les sections "aller à".

**Recherche** — Une barre de recherche en texte intégral inline limitée aux pages et à la bibliothèque de documents du site actuel. Déposez-la sur n'importe quelle page pour rendre le contenu consultable sans quitter la page.

**Bibliothèque** — Une vue consultable et recherchable de la bibliothèque de documents du site. Affiche les PDFs et fichiers téléchargés avec métadonnées, aperçu et téléchargement.

### Médias

**Image** — Télécharge ou lie une image. Supporte le texte alternatif et les légendes optionnelles. Les images sont stockées dans la médiathèque du site.

**Fichier** — Affiche une pièce jointe téléchargeable avec nom de fichier, type et taille. Stocké dans la bibliothèque de documents du site.

<Callout type="info">
  Les blocs propres à BaseBlocks, comme Répertoire, Recherche, Bibliothèque, Liens rapides et Arbre de décision, sont des extensions de l'éditeur et s'affichent dans l'éditeur comme dans le site publié.
</Callout>

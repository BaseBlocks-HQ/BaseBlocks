Sections, main and aside regions, tabs, and column slots belong to the Legacy Editor's page model. The new editor uses one ordered document and supports columns as document nodes.

<Callout type="info">
  When a legacy page opens in the new editor, sections are converted in order, two-column sections become column nodes, and aside content is placed inline. Legacy tabs are flattened into document order. Use **Legacy Editor** when you need to inspect the original layout during the beta.
</Callout>

## Section types

| Type | Columns | Region |
|---|---:|---|
| One column | 1 | Main content |
| Two columns | 2 | Main content |
| Aside | 1 | Right-side content |

Spacing is a block, not a special structural row.

## Editing sections

These controls remain available only in the Legacy Editor. The new editor hides the Sections and Blocks rail panels; use its slash menu and document controls instead.

Select a column before choosing a block from the rail. Blocks can be reordered within a column or dragged directly into another column.

## Mobile behavior

Two-column sections collapse to one column on smaller screens. Aside sections render after the main content.

L'éditeur s'ouvre lorsque vous accédez à un site. Il propose une expérience d'écriture par blocs dans le même espace de travail soigné que celui utilisé pour les pages, la publication et les paramètres.

## Zones de l'éditeur

### En-tête

La barre supérieure contient le sélecteur de site, les contrôles de partage, le raccourci vers le site publié et l'action de publication ou de retrait. L'enregistrement est automatique.

### Barre d'outils latérale

Une barre latérale flottante sur le bord gauche contient les outils de l'éditeur :

- **Pages** — l'arborescence complète des pages. Ajoutez, renommez, imbriquez et réorganisez les pages.
- **Paramètres du site** — identité, favicon, options du site publié, visibilité et domaines personnalisés.

Ajoutez des blocs avec le menu slash directement dans le canevas.

### Éditeur de page

Le canevas est le document modifiable. Tapez `/` pour insérer des blocs, sélectionnez du texte pour afficher les outils de mise en forme et déplacez les blocs compatibles pour réorganiser le document.

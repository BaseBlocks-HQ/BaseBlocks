L'éditeur s'ouvre lorsque vous accédez à un site. L'éditeur actuel propose une expérience d'écriture par blocs dans le même espace de travail soigné que celui utilisé pour les pages, la publication et les paramètres.

<Callout type="info" title="Nouvel éditeur en bêta">
  Le nouvel éditeur est en bêta pendant que nous finalisons les améliorations visuelles et comportementales. Utilisez **Éditeur classique** dans l'en-tête si une page migrée ne fonctionne pas encore comme prévu. Le contenu classique d'origine est conservé pendant cette transition.
</Callout>

## Zones de l'éditeur

### En-tête

La barre supérieure contient le sélecteur de site, l'annonce de la bêta, le bouton Éditeur classique, les contrôles de partage, le raccourci vers le site publié et l'action de publication ou de retrait. L'enregistrement est automatique.

### Rail

Une barre d'icônes flottante sur le bord gauche du canevas. Chaque icône ouvre un panneau contextuel :

- **Pages** — l'arborescence complète des pages. Ajoutez, renommez, imbriquez et réorganisez les pages.
- **Paramètres du site** — nom, identifiant, visibilité et méta description.
- **Personnalisation** — couleur d'accent, arrière-plan de l'en-tête et rayon de bordure.
- **Navigation** — style de navigation pour le site public.
- **Navigation** — choisissez l'affichage de la navigation publiée.
- **Personnalisation** — couleurs, arrondis et style de l'en-tête.
- **Paramètres du site** — identité, métadonnées, options d'en-tête, visibilité et domaines personnalisés.

Les sélecteurs de sections et de blocs propres à l'ancien éditeur sont masqués dans le nouvel éditeur. Ajoutez des blocs avec le menu slash directement dans le canevas.

### Éditeur de page

Le canevas est le document modifiable. Tapez `/` pour insérer des blocs, sélectionnez du texte pour afficher les outils de mise en forme et déplacez les blocs compatibles pour réorganiser le document. Les pages classiques existantes sont converties à l'ouverture ; le document natif est créé lors de la première modification.

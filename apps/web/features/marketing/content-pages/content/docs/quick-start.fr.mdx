Créez un espace de travail d'équipe, un site, une page, et publiez — en quelques clics.

## Workflow

<Steps>
<Step>

### Créer un compte

Rendez-vous sur [baseblocks.dev](https://baseblocks.dev) et connectez-vous avec Google, GitHub ou Microsoft. Aucune carte bancaire requise.

</Step>
<Step>

### Configurer votre espace de travail

Après la création de votre compte, vous êtes redirigé vers une page de configuration. Entrez le nom de votre équipe et choisissez un identifiant — l'identifiant se génère automatiquement à partir du nom mais vous pouvez le modifier.

Votre identifiant devient le sous-domaine de tous les sites dans cet espace de travail :

```
https://votre-identifiant.baseblocks.dev
```

L'identifiant doit être en minuscules et séparé par des tirets.

Si vous avez reçu une invitation à rejoindre une équipe existante, vous pouvez l'accepter depuis cette même page au lieu de créer un nouvel espace de travail.

</Step>
<Step>

### Créer un site

Dans le tableau de bord de votre équipe, cliquez sur **Nouveau site**. Donnez-lui un nom et un identifiant.

Votre site sera accessible à :

```
https://votre-equipe.baseblocks.dev/votre-site
```

</Step>
<Step>

### Ouvrir l'éditeur

Cliquez sur votre site pour l'ouvrir. Vous arrivez dans l'éditeur avec une première page vide prête à être construite.

L'éditeur comporte trois zones :
- **En-tête** — changement de site, annonce de la bêta, partage, raccourci vers le site publié et contrôles de publication
- **Rail** — pages, navigation, personnalisation et paramètres du site
- **Canevas** — le document par blocs. Toutes les modifications sont enregistrées automatiquement.

Le nouvel éditeur est actuellement en bêta. **Éditeur classique** dans l'en-tête ouvre l'ancien éditeur conservé si nécessaire.

</Step>
<Step>

### Ajouter du contenu

Cliquez dans la page et commencez à écrire. Tapez `/` pour ouvrir le menu des blocs, puis insérez des titres, listes, encadrés, pièces jointes, pages, diagrammes, répertoires, liens rapides, recherches, bibliothèques et autres blocs compatibles.

</Step>
<Step>

### Publier

Cliquez sur **Publier** dans l'en-tête de l'éditeur. Votre brouillon est mis en ligne immédiatement à l'URL de votre site.

C'est tout. Votre site est en ligne.

</Step>
</Steps>

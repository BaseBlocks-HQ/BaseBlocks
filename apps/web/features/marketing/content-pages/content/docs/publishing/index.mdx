## Publication status

A site is either **published** or **unpublished**. Publishing makes its pages available at the generated site URL and any verified custom domain. Unpublishing removes public access.

<Callout type="warn" title="Current publishing model">
  Published sites currently read the latest saved page content. Editing a published site can therefore change what visitors see without a separate snapshot deployment. Historical snapshots and rollback are not currently available.
</Callout>

## Publishing

<Steps>
<Step>

### Build your site

Create pages and edit content. Changes save automatically.

</Step>
<Step>

### Publish

Click **Deploy** in the editor header. The site becomes available immediately.

</Step>
</Steps>

## Unpublishing

Open the deployment menu and choose **Unpublish**. The generated URL and custom domain stop serving the site until it is published again. Saved content is not deleted.

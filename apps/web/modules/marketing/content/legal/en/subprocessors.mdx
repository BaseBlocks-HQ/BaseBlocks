## Scope

This page lists the main third-party service providers that BaseBlocks currently uses to help operate:

- the `baseblocks.dev` website
- the BaseBlocks web app
- BaseBlocks-hosted workspaces and published sites

## Current subprocessors

### Vercel

- Provider: **Vercel Inc.**
- Role: website hosting and frontend delivery
- Services used: deployment, frontend hosting, and related asset delivery

### Convex

- Provider: **Convex, Inc.**
- Role: backend infrastructure, database, and operational hosting
- Services used: application backend, data storage, and server-side operations

### Google

- Provider: **Google LLC**
- Role: identity provider
- Services used: Google sign-in where enabled by BaseBlocks

### GitHub

- Provider: **GitHub, Inc.**
- Role: identity provider
- Services used: GitHub sign-in where enabled by BaseBlocks

### Microsoft

- Provider: **Microsoft Corporation**
- Role: identity provider
- Services used: Microsoft sign-in where enabled by BaseBlocks

### S3-compatible storage providers

- Provider: **Varies by deployment**
- Role: file and asset storage
- Services used: storage and delivery of uploaded documents, media, and exported assets

## Changes

BaseBlocks may update this page from time to time as subprocessors are added, removed, or replaced.

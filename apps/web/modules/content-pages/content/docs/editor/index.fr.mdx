L'éditeur s'ouvre lorsque vous accédez à un site. Il comporte trois zones :

## Zones de l'éditeur

### En-tête

La barre supérieure. Contient le nom du site, le bouton de publication, le déploiement, le partage et les contrôles d'historique de déploiement (restauration). La sauvegarde est automatique ; la publication rend votre brouillon accessible au public.

### Rail

Une barre d'icônes flottante sur le bord gauche du canevas. Chaque icône ouvre un panneau contextuel :

- **Pages** — l'arborescence complète des pages. Ajoutez, renommez, imbriquez et réorganisez les pages.
- **Paramètres du site** — nom, identifiant, visibilité et méta description.
- **Personnalisation** — couleur d'accent, arrière-plan de l'en-tête et rayon de bordure.
- **Navigation** — style de navigation pour le site public.
- **Sections** — structures à une colonne, deux colonnes ou contenu latéral.
- **Blocs** — tous les éléments de contenu disponibles (titres, images, blocs de code, etc.).

Les contrôles d'annulation et de rétablissement se trouvent en bas du rail.

### Éditeur de page

Le rendu en direct de la page. Sélectionnez une colonne, puis ajoutez des blocs depuis le rail. Les sections et les blocs peuvent être réorganisés, et les blocs déplacés entre les colonnes. Toutes les modifications sont sauvegardées automatiquement.

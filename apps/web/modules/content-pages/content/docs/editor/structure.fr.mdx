Le contenu d'une page est un seul arbre ordonné. La plupart des blocs s'affichent directement, tandis que les blocs de structure contiennent d'autres blocs.

## Blocs de structure

| Type | Contient | Description |
|---|---|---|
| Colonnes | Une ou plusieurs colonnes | Regroupe des blocs côte à côte sur bureau et les empile sur mobile. |
| Onglets | Un ou plusieurs panneaux | Garde du contenu lié au même endroit sans créer une autre page. |
| Espaceur | Rien | Ajoute de l'espace vertical entre les blocs de contenu. |

## Travailler avec la structure

### Ajouter une structure

Ouvrez **Structure** dans le rail et choisissez des colonnes, des onglets ou un espaceur. Le bloc est inséré dans le contenu de la page courante.

### Déplacer une structure

Les blocs de structure se déplacent comme tous les autres blocs de page. Faites-les glisser vers le haut ou vers le bas du canevas pour réordonner la page.

### Supprimer une structure

Supprimer un bloc de colonnes ou d'onglets supprime aussi le contenu imbriqué.

<Callout type="warn">
  La suppression d'une structure peut être annulée avec `Cmd Z` / `Ctrl Z`, mais uniquement pendant la session en cours. Recharger la page efface l'historique d'annulation.
</Callout>

## Comportement mobile

Les colonnes se réduisent à une seule colonne sur mobile. Les onglets et les espaceurs conservent l'ordre du contenu, avec des hauteurs d'espaceur réduites sur les écrans plus petits.

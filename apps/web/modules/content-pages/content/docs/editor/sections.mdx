A page is an ordered list of sections. A section contains one or two columns, and a column can contain any number of blocks.

## Section types

| Type | Columns | Region |
|---|---:|---|
| One column | 1 | Main content |
| Two columns | 2 | Main content |
| Aside | 1 | Right-side content |

Spacing is a block, not a special structural row.

## Editing sections

Choose **Sections** in the editor rail to add one. Drag its handle to reorder it within the main or aside region. Select a section and use the delete action to remove the section, its columns, and its blocks.

Select a column before choosing a block from the rail. Blocks can be reordered within a column or dragged directly into another column.

## Mobile behavior

Two-column sections collapse to one column on smaller screens. Aside sections render after the main content.

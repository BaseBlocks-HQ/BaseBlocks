L'éditeur s'ouvre lorsque vous accédez à un site. Il comporte trois zones :

**En-tête** — la barre supérieure. Contient le nom du site, le bouton de publication, le déploiement, le partage et les contrôles d'historique de déploiement (restauration). La sauvegarde est automatique ; la publication rend votre brouillon accessible au public.

**Rail** — une barre d'icônes flottante sur le bord gauche du canevas. Chaque icône ouvre un panneau contextuel :

- **Pages** — l'arborescence complète des pages. Ajoutez, renommez, imbriquez et réorganisez les pages.
- **Paramètres du site** — nom, identifiant, visibilité et méta description.
- **Personnalisation** — couleur d'accent, arrière-plan de l'en-tête et rayon de bordure.
- **Navigation** — style de navigation pour le site public.
- **Mises en page** — tous les types de lignes disponibles pour structurer une page.
- **Blocs** — tous les éléments de contenu disponibles (titres, images, blocs de code, etc.).

Les contrôles d'annulation et de rétablissement se trouvent en bas du rail.

**Canevas** — le rendu en direct de la page. Cliquez sur une mise en page pour sélectionner un emplacement, puis ajoutez des blocs depuis le rail. Faites glisser la poignée sur le bord gauche d'une mise en page pour la réorganiser. Toutes les modifications sont sauvegardées automatiquement.

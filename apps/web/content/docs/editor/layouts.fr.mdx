Une mise en page est une ligne horizontale qui divise une page en un ou plusieurs emplacements. Chaque emplacement contient un élément. Les mises en page s'empilent verticalement pour construire une page.

## Types de mises en page

| Type | Emplacements | Description |
|---|---|---|
| Colonne unique | 1 | Ligne pleine largeur. Choix par défaut pour la plupart du contenu. |
| Deux colonnes | 2 | Colonnes de largeur égale côte à côte. |
| Trois colonnes | 3 | Trois colonnes de largeur égale. |
| Grille | 4 | Grille deux par deux d'emplacements égaux. |
| Barre latérale gauche | 2 | Emplacement étroit à gauche, emplacement large à droite. |
| Barre latérale droite | 2 | Emplacement large à gauche, emplacement étroit à droite. |
| Espaceur | 0 | Ligne vide pour l'espacement vertical. Aucun emplacement de contenu. |

## Travailler avec les mises en page

### Ajouter une mise en page

Cliquez sur **Ajouter une mise en page** dans la barre latérale, ou cliquez sur le bouton **+** qui apparaît entre les mises en page sur le canevas lorsque vous survolez. Sélectionnez le type de mise en page dans le sélecteur.

### Réorganiser les mises en page

Chaque mise en page a une poignée de glissement sur son bord gauche. Faites glisser la poignée pour déplacer la mise en page vers le haut ou vers le bas de la page.

### Supprimer une mise en page

Survolez une mise en page et cliquez sur le menu **⋯** qui apparaît à droite. Sélectionnez **Supprimer la mise en page**. Cela supprime la mise en page et tous les éléments qu'elle contient.

<Callout type="warn">
  La suppression d'une mise en page peut être annulée avec `Cmd Z` / `Ctrl Z`, mais uniquement pendant la session en cours. Recharger la page efface l'historique d'annulation.
</Callout>

## Comportement particulier des mises en page

### Mises en page espaceur

Les mises en page espaceur ajoutent de l'espace blanc vertical entre les lignes de contenu. Elles n'ont pas d'emplacements de contenu.

Hauteurs disponibles :

| Taille | Hauteur approximative |
|---|---|
| Petit | 24px |
| Moyen | 48px |
| Grand | 96px |
| Très grand | 192px |

Sélectionnez la taille lors de l'ajout de l'espaceur, ou modifiez-la depuis le menu **⋯** de la mise en page.

### Comportement mobile

Toutes les mises en page multi-colonnes se réduisent à une seule colonne sur mobile. L'ordre des emplacements sur bureau devient l'ordre de haut en bas sur mobile. Les hauteurs des espaceurs sont réduites proportionnellement sur les écrans plus petits.

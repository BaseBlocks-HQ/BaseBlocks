The editor opens when you enter a site. It has three areas:

**Sidebar** — the page tree. Add, rename, nest, and reorder pages here. Drag pages to reorganize. Also contains the element picker when you're adding content.

**Canvas** — the live page render. Click any block to select it. Drag the handle on the left side of a layout to reorder it. Edits are saved automatically.

**Element picker** — the catalogue of all available blocks and sections. Open it from the sidebar to insert an element into the selected layout slot.

Accédez aux paramètres du site depuis l'en-tête de l'éditeur → **Paramètres**, ou depuis le tableau de bord de l'équipe en cliquant sur l'icône de paramètres sur une carte de site.

## Général

**Nom** — Le nom d'affichage du site. Affiché dans le tableau de bord, l'éditeur et optionnellement dans la navigation du site. Peut être modifié à tout moment.

**Identifiant** — Le segment de chemin qui identifie ce site dans l'URL :

```
https://votre-equipe.baseblocks.dev/[identifiant-site]
```

<Callout type="warn">
  Changer l'identifiant du site change l'URL de chaque page du site. Les liens et favoris existants seront cassés. Il n'y a pas de redirection automatique.
</Callout>

**Méta description** — Une courte description utilisée dans les aperçus d'onglets du navigateur, les cartes de partage social (Open Graph) et les extraits de moteurs de recherche. Gardez-la en dessous de 160 caractères.

## Visibilité

Contrôle qui peut voir le site publié. Quatre options :

| Mode | Qui peut voir |
|---|---|
| Public | Toute personne avec l'URL |
| Privé | Membres de l'équipe connectés uniquement |
| Par lien | Toute personne avec l'URL exacte (non découvrable) |
| Mot de passe | Toute personne avec le code d'accès |

Voir [Visibilité du site](/docs/publishing/visibility) pour les détails sur chaque mode.

## Supprimer un site

La suppression du site est disponible en bas des Paramètres du site. Supprimer un site supprime définitivement toutes les pages, mises en page, documents et instantanés.

<Callout type="error">
  La suppression d'un site est permanente et ne peut pas être annulée. Il n'y a pas de chemin de récupération après la suppression.
</Callout>
